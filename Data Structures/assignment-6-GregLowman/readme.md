# Movie Collection — Binary Search Tree

A C++ BST implementation for managing a movie collection, keyed on movie name.

## Overview

- Insert movies (name, director, genre) via recursive BST insertion
- In-order traversal to display full collection alphabetically
- Search by name or filter by director
- Print only leaf nodes
- Calculate tree height

## Build

```bash
mkdir build && cd build
cmake ..
make
```

## Usage

```bash
./run_app <movies.csv>
./run_tests
```
