# Music Playlist — Singly Linked List

A C++ implementation of a music playlist using a singly linked list, with a menu-driven driver for interactive playback management.

## Overview

- Linked list of `Song` nodes (name, artists, liked status)
- Supports insertion at head or after a given song
- Search by song name or artist
- Toggle liked/unliked status per song
- Full playlist display with `->` separators

## Build

```bash
mkdir build && cd build
cmake ..
make
```

## Usage

```bash
./run_app
./run_tests
```
