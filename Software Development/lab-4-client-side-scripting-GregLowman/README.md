# Interactive Calendar — Client-Side JavaScript

A browser-based calendar application built with vanilla JavaScript, HTML, and CSS.

## Overview

- Displays a monthly calendar with navigation
- Dynamically generates dates via JavaScript — no frameworks or libraries

## Structure

```
Calendar/
  index.html
  resources/
    css/style.css
    js/script.js
```

## Usage

Open `Calendar/index.html` in any browser.
