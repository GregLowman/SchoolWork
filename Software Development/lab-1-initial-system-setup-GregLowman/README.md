# Docker + Node.js Setup

A minimal Node.js application bootstrapped with Docker Compose.

## Overview

- `docker-compose.yaml` — defines the application and database services
- `index.js` — entry point for the Node.js server
- `Submission/` — contains screenshots and notes from the initial environment setup

## Usage

```bash
docker-compose up --build
```

The application will be available at `localhost:3000`.
