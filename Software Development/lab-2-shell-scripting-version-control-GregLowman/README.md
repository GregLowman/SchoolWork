# Shell Scripting and Version Control

Bash scripting exercises alongside a C++ application, managed with Git.

## Overview

- `git_helper.sh` — shell script automating common Git workflow steps
- `main.cpp` — accompanying C++ program
- `submission/` — captured outputs from git command exercises and a pull request screenshot

## Usage

```bash
chmod +x git_helper.sh
./git_helper.sh
```
