# Personal Portfolio Website — HTML/CSS

A static personal portfolio site built with HTML and CSS.

## Overview

Pages: home, hobbies, projects, resume — all styled with a shared `style.css`.

## Structure

```
submission/
  views/         — HTML pages
  resources/
    css/         — stylesheet
    img/         — images and icons
```

## Usage

Open `submission/views/home.html` in any browser.
